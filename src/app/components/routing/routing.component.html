Routing

Click the button and watch the url to view the routing
Absolute path

redirect to root-url + path

i.e. localhost:4200/path

Note: Remember, the path must be available in the routing table

Relative path

redirect relative to current-url + path

i.e. localhost:4200/routing/path

Note: Remember, the relative path must be available in the routing table

Route Param Passing

Note: You can enter any value after the url '/routing' in the address bar and click enter to redirect to detail page.

Query Param Passing

Note: You can enter any value in the format
'?paramName1=paramValue1¶mName2=paramValue2' after the url '/routing' in the address bar and click enter to redirect to detail page.

Hash Fragment Passing

Note: You can enter any value in the format '#fargmentValue' after the url '/routing' in the address bar and click enter to redirect to detail page.
