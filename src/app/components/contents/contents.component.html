1. Installing Node.js
    node.js
  2. Installing Angular CLI
    npm install @angular/cli
  3. Creating Angular project
    ng new my-first-app
  4. Build and Serve Application
    ng serve --open
  5. Workflow
    index.html → main.ts → app.module.ts → app.component.ts → app.component.html
  6. Adding Components manually and registering in app.module
  7. Adding Components via Angular CLI
    ng generate component component-name or ng g c component-name
  8. Using the components with the selector as html element
    <app-new-component></app-new-component>

  13. Routing - Simulating multiple pages
    1. Absolute Path
      • routerLink="/path1" - Navigates to path with root-path

        In Routing table : {{routingTablePath1}}

    2. Relative Path
      • routerLink="path2" - Navigates to the path relative to current-path

        In Routing table : {{routingTablePath2}}

    3. Route Parameters
      • routerLink="path3/param1/param2" - Navigates to the path which includes param1 and param2 variables

        In Routing table : {{routingTablePath3}}

    4. Query Parameters
      • routerLink="path4" - Can add '?parameter1=value1¶meter2=value2' to the route

        In Routing table : {{routingTablePath4}}

        Complete route will be as : 'rootpath/path4?parameter1=value1¶meter2=value2'

    5. Hash Fragments
      • routerLink="path5" - Can add '#anyStringWithoutSpace' to the route

        In Routing table : {{routingTablePath5}}

        Complete route will be as : 'rootpath/path5#anyStringWithoutSpace'
